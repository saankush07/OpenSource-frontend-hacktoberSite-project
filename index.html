What Is Hacktober Fest?

Hacktober Fest is a month-long event celebrating open source software. It encourages participation in the open source community and rewards contributors.

How To Participate

  • Register on the Hacktober Fest website.
  • Contribute to open source projects by submitting pull requests.
  • Complete at least four pull requests in October to earn a t-shirt.
